Frequently Asked Questions

The answer of the common question's people asked us frequently

Flexbox are made for one dimentional layout and grid are made for two dimentional layout. You can only use flexbox row wise or column wise once at a time. On the other side grid can be use both row wise and column wise at the same time.

Bootstrap comes with a lot of pre styled kits and responsivness. You have to over-write codes of bootstrap code if you want to change the design or format of a kit.
On the other hand tailwind comes with huge number of items, more flexibility and responsivness. You have to write every line of css code to give the website design. Which give you more flexibility.

A CSS box model is the model of html eliments which shows in the web-browser.In a css box model there are a contant, some padding, border and some margin.All of the eliment's of a box model are changeable.

Semantic tag present's the meaning of an html page. It makes the html codes more meaningfull by better defining the different sections and layout of web pages.
